# 🍽️ InkaGrill - Features y Paquetes de Servicio

## 📋 Descripción General
Sistema completo de gestión para restaurantes con presencia web y panel administrativo profesional.

---

## 🎁 PAQUETES DE SERVICIO

### 📦 PAQUETE BÁSICO - "Vitrina Digital"
**Perfecto para:** Restaurantes que buscan presencia en línea y mostrar su menú

#### Features Incluidos:
- ✅ **Sitio Web Responsivo**
  - Landing page profesional con diseño moderno
  - Navegación intuitiva
  - Optimizado para desktop, tablet y móvil
  
- ✅ **Sección Menú**
  - Catálogo de platos organizado por categorías
  - Descripciones detalladas y precios
  - Búsqueda por categoría o nombre de plato
  - Interfaz limpia y fácil de usar

- ✅ **Galería de Imágenes**
  - Carousel interactivo con 6+ imágenes
  - Navegación por flechas, puntos indicadores y teclado
  - Soporte touch/swipe para móviles
  - Captions descriptivos

- ✅ **Información del Restaurante**
  - Sección "Quiénes Somos"
  - Ubicación con mapa integrado de Google Maps
  - Horarios de atención
  - Datos de contacto

- ✅ **Botón de Contacto WhatsApp Flotante**
  - Acceso rápido a WhatsApp
  - Disponible en toda la página
  - Animación de pulso

---

### 📦 PAQUETE PROFESIONAL - "Operación Completa"
**Incluye todo el Paquete Básico +**

#### Features Adicionales:

- ✅ **Sistema de Reservas**
  - Formulario de reserva en el sitio web
  - Selección de fecha, hora y cantidad de personas
  - Horarios dinámicos según el día (viernes/sábado horario extendido)
  - Validación de fechas pasadas
  - Confirmación visual de reserva
  - Notificación de error con feedback

- ✅ **Panel de Administración Seguro**
  - Autenticación de usuarios (login/contraseña)
  - Interfaz protegida
  - Diseño intuitivo y profesional

- ✅ **Gestión de Reservas**
  - Tabla de todas las reservas con detalles completos
  - Filtros por estado (Pendiente, Confirmada, Cancelada)
  - Filtro por fecha
  - Búsqueda en tiempo real por nombre, email o teléfono
  - Cambio de estado inline editable
  - Eliminación de reservas canceladas
  - Estadísticas en dashboard (Total, Pendientes, Confirmadas)

- ✅ **Gestión de Menú**
  - CRUD completo de categorías
  - CRUD completo de platos
  - Edición de precios
  - Reordenamiento de categorías y platos (drag & drop visual)
  - Búsqueda inteligente por categoría o nombre de plato
  - Interfaz amigable y rápida

- ✅ **Notificaciones en Tiempo Real**
  - Sistema de polling (cada 10 segundos)
  - Alertas de nuevas reservas
  - Banner flotante con animación
  - Auto-cierre después de 5 segundos

- ✅ **Responsive Design Avanzado**
  - Sidebar colapsable en desktop
  - Sidebar minimizado por defecto en celulares
  - Texto visible al expandir en móviles
  - Optimización completa en todos los breakpoints

## 🛠️ TECNOLOGÍAS UTILIZADAS

### Frontend
- HTML5 semántico
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript ES6+ (Vanilla, sin dependencias)
- Font Awesome 6.5.1 (iconografía)
- Google Maps Embed API

### Backend
- PHP 7.4+
- MySQL/MariaDB
- API RESTful

### Características de Arquitectura
- Diseño responsive mobile-first
- PWA-ready (offline-capable con Service Workers)
- Seguridad: hashing de contraseñas, validaciones CSRF
- Performance optimizado
- Accesibilidad WCAG 2.1
